124  Data set for checking a special function.
125 
126  Parameters
127  ----------
128  func : function
129  Function to test
130  filename : str
131  Input file name
132  param_columns : int or tuple of ints
133  Columns indices in which the parameters to `func` lie.
134  Can be imaginary integers to indicate that the parameter
135  should be cast to complex.
136  result_columns : int or tuple of ints, optional
137  Column indices for expected results from `func`.
138  result_func : callable, optional
139  Function to call to obtain results.
140  rtol : float, optional
141  Required relative tolerance. Default is 5*eps.
142  atol : float, optional
143  Required absolute tolerance. Default is 5*tiny.
144  param_filter : function, or tuple of functions/Nones, optional
145  Filter functions to exclude some parameter ranges.
146  If omitted, no filtering is done.
147  knownfailure : str, optional
148  Known failure error message to raise when the test is run.
149  If omitted, no exception is raised.
150  nan_ok : bool, optional
151  If nan is always an accepted result.
152  vectorized : bool, optional
153  Whether all functions passed in are vectorized.
154  ignore_inf_sign : bool, optional
155  Whether to ignore signs of infinities.
156  (Doesn't matter for complex-valued functions.)
157  distinguish_nan_and_inf : bool, optional
158  If True, treat numbers which contain nans or infs as as
159  equal. Sets ignore_inf_sign to be True.
160 
161  """
