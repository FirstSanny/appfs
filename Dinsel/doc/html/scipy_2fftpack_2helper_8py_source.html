12 
13  The returned float array contains the frequency bins in
14  cycles/unit (with zero at the start) given a window length `n` and a
15  sample spacing `d`::
16 
17  f = [0,1,1,2,2,...,n/2-1,n/2-1,n/2]/(d*n) if n is even
18  f = [0,1,1,2,2,...,n/2-1,n/2-1,n/2,n/2]/(d*n) if n is odd
19 
20  Parameters
21  ----------
22  n : int
23  Window length.
24  d : scalar, optional
25  Sample spacing. Default is 1.
26 
27  Returns
28  -------
29  out : ndarray
30  The array of length `n`, containing the sample frequencies.
31 
32  Examples
33  --------
34  >>> from scipy import fftpack
35  >>> sig = np.array([-2, 8, 6, 4, 1, 0, 3, 5], dtype=float)
36  >>> sig_fft = fftpack.rfft(sig)
37  >>> n = sig_fft.size
38  >>> timestep = 0.1
39  >>> freq = fftpack.rfftfreq(n, d=timestep)
40  >>> freq
41  array([ 0. , 1.25, 1.25, 2.5 , 2.5 , 3.75, 3.75, 5. ])

55  SciPy's FFTPACK has efficient functions for radix {2, 3, 4, 5}, so this

57  greater than or equal to `target`. (These are also known as 5-smooth
58  numbers, regular numbers, or Hamming numbers.)
59 
60  Parameters
61  ----------
62  target : int
63  Length to start searching from. Must be a positive integer.
64 
65  Returns
66  -------

80  >>> a = np.random.randn(min_len)
81  >>> b = fftpack.fft(a)

84  211 us, a speedup of 630 times:
